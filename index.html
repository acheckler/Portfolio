About Me

I am a soon-to-be Bloc Bootcamp graduate living in Milwaukee, Wisconsin. I have experience building responsive web-based applications. I specialize in JavaScript ES6, React, Node, and PostgreSQL.

As soon as I started working with JS in 2020, I was hooked. I grew up enamored with video games and all things tech, but I never thought I could learn how to code, mostly because I had never been exposed to it. So once a friend (and programmer) encouraged me to try it, I finally did. A few weeks later I enrolled in a boot camp, and never looked back!

I hope one day I can share my passion with young women and empower them to pursue their interests in tech, as I wish someone had done the same for me!

Frigo

An application built to help reduce food waste by keeping track of what perishables are currently in the user's fridge, and when they were purchased. A recipe search function is also included for users that need the inspiration to use what they currently have.

Built using ReactJs, NodeJs, ExpressJs, PostgreSQL, and CSS3.

Repository Live App

Content Curator

Content Curator is an app for anyone who has a topic they'd like to post about, but no content to go with it. Free to use Unsplash images and credited quotes are displayed for the user based on their search terms.

Built using JavaScript, jQuery, HTML5, and CSS3.
